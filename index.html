Hi, my name is

miguel brown

Desarrollador web y administrador de bases de datos. Me destaco por mi proactividad y capacidad para resolver problemas complejos. Domino tecnologías como PHP, JavaScript, MySQL, Git y Linux. Puedo crear/administrar aplicaciones y bases de datos personalizadas, optimizar código y supervisar sistemas para garantizar su óptimo funcionamiento. En mi experiencia laboral he desarrollado aplicaciones internas, automatizado procesos de respaldo de BBDD y configurado herramientas para importar/exportar datos eficientemente. Además, tengo experiencia en soporte técnico, habiendo trabajado resolviendo problemas en smartphones y configurando redes. También poseo excelentes habilidades blandas, obtenidas a través de experiencias como el voluntariado en Python Chile. Mi orientación principal es el backend, específicamente la administración de bases de datos y optimización de consultas SQL.
